# Euskara App

This is an app (really just a single HTML page) to practice the vocabulary and
translation exercises of this Euskara course:

https://kondaira.net/ikastaroa

It allows you to select a range of lessons to practice, and also to decide
whether to translate from basque to your language (really, no enforcement or
validation) or from spanish/catalan to basque.

This is as simple as an app can be, just to have all the lessons examples
available for practicing on the go.

I have added though some examples of real world text, which I have translated
to catalan with ChatGPT.
